Jellypy — Jellyfin plugin for Sonarr & Radarr

Event-driven plugin for Jellyfin integrating with Sonarr and Radarr. Built in C# (.NET 8) and containerised with Docker.

What it does

Jellypy listens for media events from Jellyfin and performs integration tasks with self-hosted media managers such as Sonarr and Radarr. Typical uses are automatic tagging, library refresh and notifying downstream managers when media is available.

Quick integration example

Example webhook payload (pseudo):

curl -X POST https://your-jellypy.example/api/webhook \
  -H "Content-Type: application/json" \
  -d '{"event":"downloadComplete","source":"sonarr","title":"My.Show.S01E02"}'

Installation

  1. Clone the repository.
  2. Build with .NET 8 and optionally containerise with Docker.
  3. Configure the webhook endpoint in Sonarr/Radarr to notify Jellypy on completed downloads.

See the GitHub repo for source, releases and detailed configuration examples.
