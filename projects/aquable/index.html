Aquable

Control your Chihiros devices via Bluetooth

About

Aquable is a BLE controller for Chihiros aquarium devices. Whether you're running Home Assistant, Docker, or a standalone Python service, you can manage your LED lights, dosing pumps, and terrarium gear from one place.

Status: In active development at an early stage. Expect new features and refinements soon.

GitHub Repository Report Issues

Supported Devices

  • Chihiros 4 Head Dosing Pump
  • Chihiros WRGB II
  • Chihiros WRGB II Pro
  • Chihiros LED A2
  • Chihiros Tiny Terrarium Egg
  • Chihiros C II (RGB, White)
  • Chihiros Universal WRGB
  • Chihiros Z Light TINY
  • + other LED models (untested)

Built on the open-source Chihiros LED Control project. Licensed under MIT.

How to Run It

Pick the deployment method that works best for you:

Home Assistant Add-on (coming soon)

Install from the HACS add-on store. Easy, integrated, recommended.

Auto-configured Bluetooth, persistent storage, dashboard integration

Docker (coming soon)

Multi-architecture support (amd64, arm64, armv7). Perfect for self-hosting or Unraid.

Health monitoring, auto-restart, volume persistence

Python Service

Direct install for development, advanced users, or Raspberry Pi.

Native performance, full control, ideal for tinkering
