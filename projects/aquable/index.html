AquaBle

Control your Chihiros devices via Home Assistant

AquaBle is a lightweight Home Assistant add-on that gives you control over your Chihiros aquarium devices directly from the Home Assistant interface. Manage your LED lights, dosing pumps, and terrarium gear seamlessly with integrated Ingress support—no port forwarding or complex configuration needed.

Status: In very active development at an early stage. Expect new features and refinements soon.

GitHub Repository Report Issues

Supported Devices

  • Chihiros LED Aquarium Lights
  • Chihiros Dosing Pumps

Built on the open-source Chihiros LED Control project. Licensed under MIT.

Installation

AquaBle is distributed as a Home Assistant add-on and is installed directly from a custom repository.

  1. In Home Assistant, go to Settings → Add-ons & integrations → Add-ons
  2. Click the three-dot menu and select Repositories
  3. Add the custom repository: https://github.com/caleb-venner/aquable
  4. Go to the Add-ons store and search for AquaBle
  5. Click Install and then Start
  6. Click OPEN WEB UI to access the interface via Home Assistant Ingress

Home Assistant Ingress

AquaBle is built for Home Assistant Ingress integration:

  • Access the web interface directly through the Home Assistant UI
  • No port forwarding or authentication configuration needed
  • Seamless user experience within Home Assistant
  • Works perfectly on Home Assistant mobile apps
