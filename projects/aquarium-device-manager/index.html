Aquarium Device Manager

Full-stack application to interface with aquarium hardware (sensors, pumps, Chihiros controllers) via Bluetooth LE. Backend: Python (FastAPI). Frontend: TypeScript. Containerised with Docker.

What it does

Provides an API and UI to monitor and control aquarium devices. Designed to integrate with Home Assistant or other home automation platforms.

Home Assistant example (REST)

# Example: use REST sensor in Home Assistant to fetch sensor data
rest:
  - resource: http://your-aquarium.example/api/sensors
    sensor:
      - name: "Chihiros Water Temp"
        value_template: "{{ value_json.temperature }}"

Installation

  1. See the GitHub repo for source and instructions.
  2. Run the FastAPI service, configure BLE permissions, and optionally run with Docker for ease of deployment.
